\# DevOps CI/CD Project - Booking Application



\## Project Overview

This project demonstrates a complete CI/CD pipeline using:

\- \*\*Git/GitHub\*\*: Version control and branching strategy

\- \*\*Jenkins\*\*: Continuous Integration and Deployment

\- \*\*Docker\*\*: Containerization



\## Pipeline Structure

\- Pipeline 1: PR Testing (on Pull Request to dev)

\- Pipeline 2: Build \& Deploy (on push to dev)

\- Pipeline 3: Versioned Build (on tag vX.Y.Z)



\## Branch Strategy

\- main: Protected production branch

\- dev: Development branch

\- feature/\*: Feature branches



\## Technologies

\- React 19 + Vite

\- Bootstrap 5

\- Node.js

\- Docker
